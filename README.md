# MyFFTCG
Lightweight FFTCG client

See visual starter guide at: https://fftcg.cdn.sewest.net/2020-02/fftcgrulesheet-en.pdf

See advanced ruleset at: https://fftcg.cdn.sewest.net/2022-11/fftcg-opus-cr-english-20220811.pdf

The ultimate goal for this application will be to browse a card library, construct a deck of 50 cards, and play against someone else with the application facilitating the flow of the game, tracking the game state and resolving card interactions.
