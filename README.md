# MyFFTCG
Lightweight FFTCG client

Includes a deck manager, full card browser, and a game board.

Links to rules and guides are found in the Help menu.

<img width="1900" height="1065" alt="image" src="https://github.com/user-attachments/assets/750c6c6e-aa8f-42d2-8199-ed6ca0a2c879" />

The ultimate goal for this application will be toplay against someone else with the application facilitating the flow of the game, tracking the game state and resolving card interactions.
